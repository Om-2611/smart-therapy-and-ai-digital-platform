STAAD — Product Requirements Document
 (PRD)
 
Product Overview
STAAD is a modern online therapy platform designed specifically for therapists working with SLD, ADHD,
Anxiety, Depression, Intellectual Disability, Dyslexia, and Autism.
The platform combines secure live video conferencing, interactive therapy modules, real-time collaborative
activities, session tracking, and a plugin-based therapy module system.
Core USP:
During live therapy sessions, both therapist and client can simultaneously interact with therapy modules
like Maze, Bubble Splash, Talking Calculator, Matching Games, Attention Tasks, and Memory Games.
Primary Goal
Build a production-ready MVP using mostly free/serverless tools with clean architecture, scalable plugin
system, responsive UI, minimalistic design, calm therapy-focused aesthetics, and maintainable codebase.
Design Language
- Calm sage green palette
- Minimal text-heavy UI
- Large breathing spaces
- Rounded corners
- Soft shadows
- Warm off-white background
- DM Sans + DM Serif typography
- Emotionally calming interface
